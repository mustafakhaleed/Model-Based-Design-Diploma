# Model-Based Design (MBD) Diploma Portfolio

This repository contains my complete practical work, labs, and projects completed during my Model-Based Design Diploma. The curriculum focuses on developing, simulating, and deploying control algorithms for embedded systems (Automotive focus) using the MATLAB/Simulink toolchain.

## 🛠️ Tech Stack & Tools
* **Software:** MATLAB / Simulink / Stateflow (Version R202Xb)
* **Code Generation:** Embedded Coder (C target)
* **Hardware Interfacing:** Arduino Support Package for Simulink
* **Standards:** AUTOSAR Application Design concepts

## 📂 Repository Structure & Modules
1. **01_MBD_Concepts:** Introduction to the V-Cycle development process.
2. **02_MATLAB_Basics_and_Scripting:** Core syntax, data types, and automation scripts.
3. **03_Simulink_Basics_and_Architecture:** Signal routing, subsystems, bus objects, and model referencing.
4. **04_Arduino_Simulink_Interface:** Hardware-in-the-loop (HIL) concepts and deploying models to AVR microcontrollers.
5. **05_PID_Controller_Design:** Closed-loop control systems, tuning parameters, and stability analysis.
6. **06_System_Modeling:** Transfer functions, physical modeling, and plant dynamics.
7. **07_Stateflow_Design_Module:** Finite State Machines (FSM), reactive logic, and supervisory control.
8. **08_Embedded_Coder_and_C:** Auto-generating MISRA-C compliant code from Simulink models.
9. **09_BMS_Capstone_Project:** Advanced Battery Management System featuring SOC estimation, fault management, and AUTOSAR compatibility.
